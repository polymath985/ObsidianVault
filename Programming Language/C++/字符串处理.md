# 字符串处理函数

- [[string]]转换为int,float,double

```
 string str = "100";
 
 //using namespace std
 int INum = stoi(str);
 float FNum = stof(str);
 double DNum = stod(str);

 //注意，出现不合法转换会抛出异常
```

- 其他类型值转string

```
 int a = 1;
 float f = 2.0;
 double d = 3.14;

 //using namespace std
 string Istr = to_string(a);
 string Fstr = to_string(f);
 string Dstr = to_string(d);
```

- 判断一个char是否为字母，判断大小写及其转换

```
char c = 'a';
//using namespace std
//判断是否为字母
if(isalpha(c)){}
//判断是否为大写字母
if(isupper(c)){}
//小写
if(islower(c)){}
//相互转换
toupper(c);
tolower(c);
```

# 注意事项

- 在使用getline读取某行的字符串时，需要先忽略之前输入的不需要的数据

```
int n;
cin>>n;
cin.ignore(numeric_limits<streamsize>::max(),'\n');
vector<string> strVec;
for(int i = 0 ;i < n;i++)
{
	string str;
	getline(cin,str);
	strVec.push_back(str);
}
```


> [!NOTE] cin.ignore();
> 忽略缓冲区的内容，默认忽略最后一个内容
> 第一个参数：忽略的最大数量int
> 第二个参数：忽略到的字符char
> 缓冲区剩余字符的最大容量
> `numeric_limits<streamsize>::max();`
